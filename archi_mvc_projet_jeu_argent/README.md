Nous sommes en présence du projet nsi du second trimestre. Ce projet consiste à utiliser des donnés sur un thème choisi avec pandas puis de les représenter dans des fénêtre grace à tkinter. Après une courte discussion nous nous mettions d'accord sur le sujet qui fut "les jeux d'argents". Nous avons dirigé ce programme afin de prouver que nous perdions toujours aux jeux d'argent aléatoire.

Les difficultés rencontrées dans ce projet :

La familiarisation avec les organismes tkinter et pandas ne se fit pas immédiatement.
L'organisation de nos idées fut préalablement confute
La nouvelle structure imposée pour le projet nous mis dans un profond vide mais elle fut utilie pour un programme plus organisé et compréhensible
